
OnConnection returns the connection id.

    id

+ to subscribe to a channel, returns the channel id.

    +somechannel
    <channel id>

Unsubscribes from a channel with his name or id, returns the channel id.

    -somechannel
    -23
    <channel id>

Sends a message to a channel using his name or id, returns the message id.

    !somechannel <value>
    !23 <value>

    <message id>

OnSubscription returns the value sent.

    <channel id> <message id> value
